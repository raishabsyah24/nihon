# Nihon e Ikitai Logo Drafts

Draft aset ini belum diterapkan ke admin atau mobile.

## File

- `nihon-e-ikitai-mark.svg` - brand mark utama.
- `nihon-e-ikitai-favicon.svg` - draft favicon sederhana.
- `nihon-e-ikitai-admin-logo.svg` - logo horizontal untuk admin Next.js.
- `nihon-e-ikitai-app-icon.svg` - draft icon launcher mobile.
- `nihon-e-ikitai-logo-preview.svg` - preview semua aset dalam satu kanvas.

## Arah visual

Konsepnya memakai buku terbuka, jalur menuju Jepang, matahari merah, dan aksen gold. Warna mengikuti design token proyek:

- Primary: `#659287`
- Secondary: `#9CBCB0`
- Soft bg: `#E6F0DD`
- Text dark: `#263238`
- Accent: `#D79A2B`
- CTA red: `#B4232A`

Kalau draft ini disetujui, langkah berikutnya adalah export ukuran produksi untuk favicon, Next.js public asset, dan Flutter launcher icon.
